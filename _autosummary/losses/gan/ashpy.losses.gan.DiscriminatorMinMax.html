The min-max game played by the discriminator.

\[L_{D} = - \frac{1}{2} E [\log(D(x)) + \log (1 - D(G(z))]\]

Returns the discriminator inputs. If needed it uses the encoder. The current implementation uses the number of inputs to determine whether the discriminator is conditioned or not. :param context: context for GAN models :type context: ashpy.contexts.gan.GANContext :param fake_or_real: discriminator input tensor, it can be fake (generated) or real :type fake_or_real: tf.Tensor :param condition: discriminator condition (it can also be generator noise) :type condition: tf.Tensor :param training: whether is training phase or not :type training: bool
